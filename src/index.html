Buy A Pixel

  1. Log into metamask
  2. Click on a pixel to get it's coordinates.
  3. Select the color you want
  4. Set a price (+1 finney more then current price)
  5. Buy the pixel
  6. Press 'Load Pixels' to see your own pixel
